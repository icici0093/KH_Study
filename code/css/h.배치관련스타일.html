화면 배치 방법 변경

박스 요소와 인라인 요소의 속성을 변경하여 배치 방식을 변경할 수 있다. (display 속성)

블럭요소를 인라인요소로 변경

>> display:inline 테스트
인라인요소로 변경되면서 width와 height속성이 무시된다.
첫번째 영역
두번째 영역
세번째 영역
네번째 영역
다섯번째 영역
>> display:inline-block 테스트
영역은 인라인 요소로 변경되지만 내용은 블록요소로 지정되어 width와 height속성을 설정할 수 있다.
첫번째 영역
두번째 영역
세번째 영역
네번째 영역
다섯번째 영역

인라인요소를 블럭요소로 변경

>> display:block 테스트
인라인요소를 블럭요소로 변경하면 width와 height값을 지정할 수 있고 block요소로 변경이 되는 것이다.
