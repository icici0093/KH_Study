text-decoration 속성 : 텍스트에 줄을 긋거나 줄을 없앨 때 사용한다.

 선택자 {text-decoration : none | underline 
        | overline | line-through; }의 형식으로 표현한다.
  • 하이퍼링크 밑줄 없애기
  • 텍스트영역 아래 줄 긋기
  • 텍스트영역 중간에 줄 긋기

  • 텍스트영역 위에 줄 긋기

text-transform 속성 : 영문 텍스트의 대소문자 변환시 사용한다.

 선택자 { text-transform : none | capitalize 
    | uppercase | lowercase; }의 형식으로 표현한다.
  • none
  • capitalize : 영문자의 첫 글자만 대문자로
  • uppercase : 모든 영문자를 대문자로
  • LOWERCASE : 모든 영문자를 소문자로
