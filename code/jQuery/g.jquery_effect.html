jquery 기본 효과

show() : 문서 객체를 보여줌
hide() : 문서 객체를 사라지게 함
toggle() : show()와 hide()를 번갈아 실행
slideDown() : 문서 객체를 슬라이드 효과와 보여줌
slideUp() : 문서 객체를 슬라이드 효과와 사라지게 함
slideToggle : sildeDown()과 slideUp()을 번갈아 실행함
fadeIn() : 문서객체를 선명하게 하며 보여줌
fadeOut() : 문서객체를 흐리게 하며 사라지게 함
fadeToggle() : fadeIn()과 fadeOut()을 번갈아 실행함.
